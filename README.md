<h1>🏆 Interclasse ETEC 2026</h1>

Sistema web desenvolvido para acompanhar o Interclasse da ETEC Antonio Devisate – 2026, permitindo visualizar partidas, horários, modalidades e informações gerais do evento esportivo entre turmas.

<h2>Objetivo do Projeto</h2>

O objetivo do site é centralizar as informações do interclasse em uma única plataforma, permitindo que alunos e organização acompanhem:

<li>
  Jogos do dia
</li>
<li>
  Modalidades esportivas
</li>
<li>
  Estatísticas do evento
</li>
<li>
  Informações gerais do campeonato
</li>
<li>
  Confrontos e placares
</li>

<br>

Além disso, o sistema inclui um modal de login, permitindo que organizadores tenham acesso a funções administrativas, como registrar placares.
